\# Breakwater



Breakwater is a personal job-discovery and review tool for identifying

realistic early-career opportunities.



The project focuses on:



\- Remote jobs open to Canadian applicants

\- Nearby hybrid opportunities

\- Required versus preferred experience

\- Seniority detection

\- Explainable job classifications

\- A focused daily review queue



The first milestone is a local Django application using PostgreSQL and

one validated job source.



See \[docs/master-plan.md](docs/master-plan.md) for the full product and

implementation plan.
